Compact snapshot hunks with identical added and removed lines into exact change
families that can be reviewed, accepted, or rejected as one safe decision in
the UI and CLI. Source review now includes imports, owning suites, and linked
suite/test hooks around the focused test. The new `vsnap families` command and
bundled agent skill make family-first review the default token-efficient
workflow for automated snapshot maintenance.
